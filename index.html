Microsoft Program Synthesis using Examples SDK

A framework for automatic programming or data wrangling from input-output examples.

Program Synthesis Framework

Microsoft PROSE SDK is a framework of technologies for programming by examples – automatic generation of programs from input-output examples at runtime.

Given a domain-specific language (DSL) and some input-output examples for the desired programâ€™s behavior, PROSE algorithms synthesize a ranked set of DSL programs that are consistent with the given examples.

Data Wrangling DSLs

PROSE SDK includes a pre-defined suite of technologies for various kinds of data wrangling – cleaning and pre-processing raw semi-structure data into a form amenable to analysis. These include:

  • Flash Fill, a technology for text transformation by examples, available in Microsoft Excel and PowerShell.
  • Data extraction from text files by examples, available in PowerShell and Azure Log Analytics.
  • Data extraction and transformation of JSON files by examples.
  • Predictive file splitting technology, which discovers structured columns in a text file without any examples or interaction.
  • Clustering of a set of strings into buckets by syntactic similarity.

In addition, PROSE supports compilation of the programs from all aforementioned DSLs into self-sufficient Python/R scripts.

Installation

1. Install our NuGet package 

Install-Package Microsoft.ProgramSynthesis
and

2a. Start a new DSL using Yeoman 

$ npm install -g yo generator-prose
$ yo prose
or

2b. Build on top of the existing DSLs 

using Microsoft.ProgramSynthesis.Extraction.Text;
using Microsoft.ProgramSynthesis.Extraction.Json;
